# Pasta Public

Para adicionar a imagem do banner DPSP:

1. Coloque a imagem com o nome `banner-dpsp.jpg` nesta pasta
2. A imagem será exibida automaticamente na toolbar fixa
3. Se a imagem não for encontrada, será exibido um fallback com o design DPSP

Formato recomendado:
- Nome: `banner-dpsp.jpg` (ou `.png`)
- Dimensões: 1920x80px ou proporção similar (banner horizontal)
- Formato: JPG ou PNG
